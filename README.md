# React Boilerplate

Lightweight boilerplate for React with Redux, Babel and Webpack.

## Quick Start

### Install dependencies:

```bash
$ npm install
```

### Start the server:

For desktop
```bash
$ npm start
```

For mobile
```bash
$ DEVICE=mobile npm start
```

### Build for production:

For desktop
```bash
$ npm run publish
```

For mobile
```bash
$ DEVICE=mobile npm run publish
```
